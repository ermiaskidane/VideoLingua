## Set Up Environment

- bunx create-next-app@latest
- bunx --bun shadcn@latest init
- bunx --bun shadcn@latest add
- bun add next-themes
- bun add better-auth
- bunx prisma init
- bun add prisma
- bun add @prisma/client
- bunx prisma generate
- bunx @better-auth/cli generate
- bunx prisma db push
- bunx prisma studio
